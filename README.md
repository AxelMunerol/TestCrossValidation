Le modèle exporté ici est créé dans le notebook jupyter "recherche modele" et prédit la possibilité ou non de décrocher un pret.
Le modèle présente de bons résultats pour prédire si un dossier sera validé mais beaucoup moins s'il sera refusé. Un axe d'amélioration serait d'améliorer le data set pour égaliser les fréquences d'admissions ou non.
Le smoot n'ayant rien donné il reste à balance d'une autre manière les data mais le egb modèle donne 95% de précision avec une probabilité de faux négatif
